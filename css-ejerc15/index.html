Tipos de selectores que entran en examen

Clase (class=””)

Selecciona todos los elementos con la etiqueta clase="" y les aplica los cambios designados. Un ejemplo es este parrafo, que cambia de color al ponerle la etiqueta.

Identificador (id=””)

Selecciona los elementos con la etiqueta id="" y les aplica cambios. Este parrafo cambia su color gracias a este selector.

Todos los elementos (*)

Selecciona todos los elementos que no tengan otro selector asignado previamente.

Varios elementos (h1, h2)

Selecciona todos los elementos que esten dentro de un rango seleccionado, por ejemplo los "h1" y los "h2"

Elementos contenidos en otros elementos (ol li)

Selecciona los elementos con la etiqueta "li" que esten dentro de una etiqueta "ol", por ejemplo:

  1. patatas
  2. aceite
  3. sal

Sobrevuleo (p:hover)

Hace que cambie el estilo al pasar el cursor por encima de las partes con la etiqueta.
